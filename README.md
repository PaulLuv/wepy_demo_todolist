# wepy-demo-todolist

## Usage
--------------------

### Start

```bash
git clone https://github.com/PaulLuv/wepy_demo_todolist.git

cd wepy_demo_todolist

npm install wepy-cli -g

npm install

npm run dev
```

> Remind

- 使用微信开发者工具新建项目，本地开发选择 dist 目录
- 微信开发者工具 -> 项目
  - 关闭 ES6 转 ES5
  - 关闭 代码压缩上传
  - 关闭 上传代码时样式文件自动补全
  - 开启 开发环境不校验请求域名、TLS版本以及HTTPS证书

### Build

```bash
npm run build
```
